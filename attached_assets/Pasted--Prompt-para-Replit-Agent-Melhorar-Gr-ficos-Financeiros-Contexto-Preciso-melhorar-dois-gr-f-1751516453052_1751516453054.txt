# Prompt para Replit Agent: Melhorar Gráficos Financeiros

## Contexto
Preciso melhorar dois gráficos em um aplicativo financeiro React que estão com problemas visuais significativos de usabilidade e apresentação.

## Gráfico 1: Despesas por Categoria (Gráfico de Pizza/Rosquinha)

### Problemas Atuais:
- O gráfico está muito pequeno, dificultando a visualização das fatias
- A legenda está desorganizada com muitos itens quebrados em várias linhas
- As cores são pouco contrastantes e difíceis de distinguir
- Não há interatividade ou tooltips informativos
- O layout geral parece amador e pouco profissional

### Melhorias Desejadas:
- Aumentar significativamente o tamanho do gráfico para melhor visualização
- Reorganizar a legenda em um layout grid limpo e responsivo
- Implementar uma paleta de cores moderna e vibrante com bom contraste
- Adicionar interatividade com efeitos hover e tooltips detalhados mostrando valores e percentuais
- Criar um design moderno com sombras, bordas arredondadas e tipografia elegante
- Incluir um resumo total das despesas na parte inferior

## Gráfico 2: Despesas por Contrato (Gráfico de Barras)

### Problemas Atuais:
- Os rótulos do eixo X (nomes dos contratos) estão sobrepostos e ilegíveis
- O design é muito básico e sem apelo visual
- Não há informações adicionais ou contexto sobre os dados
- As barras são simples demais, sem interatividade

### Melhorias Desejadas:
- Rotacionar os rótulos do eixo X em 45 graus para evitar sobreposição
- Implementar tooltips que mostrem o nome completo do contrato e valores detalhados
- Adicionar efeitos hover nas barras para melhor interação
- Incluir estatísticas resumidas como total geral, maior despesa e média por contrato
- Criar um design mais profissional com cores modernas e espaçamento adequado
- Garantir que seja totalmente responsivo para diferentes tamanhos de tela

## Especificações Técnicas:
- Usar React com a biblioteca Recharts para os gráficos
- Implementar Tailwind CSS para estilização moderna
- Garantir responsividade completa (mobile-first)
- Usar uma paleta de cores consistente: azuis (#3B82F6), verdes (#10B981), laranjas (#F59E0B), vermelhos (#EF4444), roxos (#8B5CF6)
- Implementar animações suaves e transições elegantes
- Adicionar sombras e bordas arredondadas para um visual moderno

## Resultado Esperado:
Dois componentes React profissionais e modernos que transformem gráficos básicos em visualizações de dados atraentes, interativas e altamente funcionais, adequadas para um dashboard financeiro corporativo.

Crie os componentes necessários e um exemplo de implementação no App.js principal.
